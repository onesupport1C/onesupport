# Сайт-визитка onesmaster.ru

Одностраничный сайт (визитка) в стиле «яркий стартап»: градиенты, тёмный фон,
анимации появления. HTML/CSS/JS + PHP-обработчик формы.
Хостится на виртуальном хостинге Reg.ru с доменом onesmaster.ru.

---

## 📁 Файлы проекта

| Файл | Назначение |
|---|---|
| `index.html` | Главная (и единственная) страница |
| `styles.css` | Стили (палитра — в блоке `:root` сверху) |
| `script.js` | Мобильное меню, анимации, отправка формы |
| `send.php` | Обработчик формы → письмо на почту |
| `.gitignore` | Исключает рабочие 1С-файлы из Git |

---

## 🚀 Как запустить локально

Просто открой `index.html` двойным кликом — сайт откроется в браузере.

Или подними локальный сервер (чтобы форма/анимации работали точнее):

```bash
# вариант 1: Python (есть почти везде)
python -m http.server 8000

# вариант 2: Node
npx serve
```

Затем открой `http://localhost:8000`.

## ✏️ Что поменять под себя (главное!)

Открывай `index.html` в любом редакторе (VS Code, Notepad++) и меняй заглушки:

| Что | Где в `index.html` | Сейчас (заглушка) |
|---|---|---|
| Название бренда | `.logo__text`, футер | `[БРЕНД]` |
| Telegram | `href="https://t.me/ВАШ_ТЕЛЕГРАМ"` | `ВАШ_ТЕЛЕГРАМ` |
| E-mail | `href="mailto:..."` и текст | `you@example.com` |
| Телефон | `href="tel:..."` и текст | `+7 900 000-00-00` |
| Заголовок hero | `.hero__title` | можно подкрутить под себя |
| Цифры в статистике/кейсах | `.stat__num`, `.case__num` | примерные — подставь свои реальные |
| Title/description (вкладка + поисковики) | `<title>`, `<meta name="description">`, OG-теги | текст-описание |

- `[БРЕНД]`

## 🎨 Как сменить палитру

Все цвета собраны в одном месте — `styles.css`, блок `:root` в самом верху:

```css
:root {
  --bg: #0B0B1A;        /* основной фон */
  --accent: #8B5CF6;    /* фиолетовый — главный акцент */
  --accent-2: #3B82F6;  /* синий — второй цвет градиента */
  --accent-3: #EC4899;  /* розовый — подсветка */
}
```

Поменял 3 значения → весь сайт перекрасился.

## 📨 Подключить форму (чтобы заявки реально приходили)

Сейчас форма показывает сообщение «отправлено», но никуда не шлёт. Чтобы заявки
приходили в твой Telegram — есть два простых пути:

### Вариант А: через Formspree (проще всего)
1. Зарегистрируйся на https://formspree.io (бесплатно, до 50 заявок/мес).
2. Создай форму, получи `form ID` (вида `xyzbqrol`).
3. В `index.html` в теге `<form>` добавь атрибуты:

```html
<form class="form" id="contactForm"
      action="https://formspree.io/f/ТВОЙ_ID" method="POST">
```

4. В `script.js` замени блок отправки (после проверки) на:

```js
fetch(form.action, {
  method: 'POST',
  body: new FormData(form),
  headers: { 'Accept': 'application/json' }
}).then(() => {
  note.textContent = 'Спасибо! Заявка отправлена.';
  form.reset();
});
```

### Вариант Б: через Telegram-бота
1. Создай бота через @BotFather, получи токен.
2. Узнай свой `chat_id` (через @userinfobot).
3. В `script.js` отправляй `fetch` на:
   `https://api.telegram.org/bot<ТОКЕН>/sendMessage?chat_id=<ID>&text=...`

Заявка будет падать прямо в твой личный Telegram.

> ⚠️ Вариант Б отправляет токен в открытом виде — подходит для визитки с
> невысокой нагрузкой. Для серьёзного трафика лучше Formspree или свой бэкенд.

## 🌐 Как выложить в интернет БЕСПЛАТНО

### GitHub Pages (рекомендация)
1. Зарегистрируйся на https://github.com.
2. Создай репозиторий, например `mysite`.
3. Загрузи туда 4 файла: `index.html`, `styles.css`, `script.js`, `README.md`.
4. Settings → Pages → Branch: `main` / root → Save.
5. Через минуту сайт будет доступен по адресу:
   `https://ТВОЙ_ЛОГИН.github.io/mysite/`

### Netlify (перетаскиванием)
1. Открой https://app.netlify.com/drop.
2. Перетащи папку с файлами в окно.
3. Получишь готовую ссылку за 30 секунд.

### Привязка своего домена (например, `имя.рф`)
- На GitHub Pages: Settings → Pages → Custom domain.
- На Netlify: Domains → Add domain.
- Домен покупается на Reg.ru / Ru-Center (`.ru` ~ 200 ₽/год).
